Задание 1.

Создать приложение на базе модального диалогового окна.
Добавить четыре кнопки.
Реализовать события клика по кнопкам по следующему алгоритму:
1. Отрпавить синхронное сообщение (PostMessage) текущему окну WM_QUIT
2. Отрпавить асинхронное сообщение (SendMessage) текущему окну WM_QUIT
3. Отрпавить синхронное сообщение текущему окну WM_DESTROY
4. Отрпавить асинхронное сообщение текущему окну WM_DESTROY

Запустить приложение. В Spy++ проанализировать сообщения которые генерируются
при нажатии каждой кнопки.
Сделать вывод о поведении программы при отрпавке синхронных и асинхронных
сообщений.

Задание 2.

Написать приложение на базе модального диалогового окна.
Реализовать функционал программного добавление окна STATIC.
Первый клик левой кнопкой мыши определяет первую точку диагонали
прямоугольника статика, отпускание кнопки мыши - вторую точку.
Не создавать статики меньше 10x10 пикселей.

Базовый функционал: 
	- статик создается при перемещении мыши с верхней левой точки
          вправо вниз
	- правый клик по статику приводит к удалению статика

Опция 1.
	- статики создаются по всем направлениям

Опция 2.
	- в правый нижний угол вывести статик с изображением корзины
	- статики можно преремещать левой кнопкой мыши в пределах окна
	- перемещение статика в корзину приводит к удалению статика
	  после отпускания левой клавиши мыши

Опция 3.
	- при перемещении статика в корзину (при зажатой левой кнопки мыши)
	  крышка корзины открывается
        - после удаления статика крышка корзины закрывается
